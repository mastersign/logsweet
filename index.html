CLI Usage¶

logsweet¶

A suite with a variety of tools for handling log messages

logsweet [OPTIONS] COMMAND [ARGS]...

Commands

broadcast

Follow logfiles and publish new lines via…

listen

Listen to text lines from ZeroMQ SUB sockets

mock

Generate random log file entries

logsweet broadcast¶

Follow logfiles and publish new lines via ZeroMQ PUB socket

logsweet broadcast [OPTIONS] FILE_GLOB

Options

-b, --bind-address <bind_address>¶

IP and port to bind the ZeroMQ socket

-a, --all-lines¶

Broadcast all existing content before following

-t, --tail-lines <tail_lines>¶

Number of tail lines to broadcast before following

-e, --encoding <encoding>¶

Encoding for reading the text files.

-s, --silent¶

Do not print new lines to the console.

-n, --name <name>¶

The source name for the watched logs.

Arguments

FILE_GLOB¶

Required argument
